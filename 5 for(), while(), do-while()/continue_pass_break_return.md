## continue / pass / break / return 차이

- ***continue***
    
    continue의 **아래 행을 수행하지 않고** 반복문의 **다음 턴**으로 넘어간다.
    
- ***pass***
    
    실행할 코드가 없음을 의미한다.
    
- ***break***
    
    해당 break를 감싼 **가장 작은 반복문**을 탈출한다. (반복문 종료)
    
- ***return***
    
    인수 없이 사용하면 **함수 실행을 종료**하고 **함수를 호출한 곳으로 돌아간다.**
