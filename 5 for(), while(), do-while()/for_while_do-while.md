## for문

> **for문 구조**
> 

```java
// int i=0   i<10    i++
for(초기값; 조건식; 증감량) {
	반복할 문장
}
```

## while문

> **while문 구조**
> 

```java
while(조건식) { // 조건식이 참일 때 반복
	반복할 문장
}
```

> **무한루프**
> 

```java
while(true) {
	반복할 문장
}
```

## do-while문

> **do-while문 구조**
> 

```java
do{
	반복할 문장 // 무조건 실행
}while(조건식);
```
